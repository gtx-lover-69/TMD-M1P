# TMD-M1P
The first ever project in the TMD lineup, created for Hack Club Blueprint. M1P stands for Music 1 Prototype.

TMD-M1 is a magnetically modular music player. Using magnets on the sides of the device, one can quickly snap on and off their favourite accessories.

TMD-M1P currently has no magnets (which kinda defeats the purpose) but it is still a work in progress 

The proposed TMD-M1 should arrive at your doorstep pre-built and pre-flashed. An open-ended developer edition is also planned.

TMD-M1 can also be assembled by those who would like. This is as simple as a little soldering, as everything is accurately marked on the PCB. The file may be flashed using Arduino IDE or any other compatible software.
